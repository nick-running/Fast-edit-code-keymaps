# Fast-edit-code-keymaps
Fast keymaps for jetbrains IDE. allows you fast development in coding
## Keyboard Shortcut
 ### Editor Tabs(编辑器标签)
  - Split Vertically(垂直分离编辑标签) `trl+Alt+Shift+|`
  - Split Horizontally(水平分离编辑标签) `trl+Alt+Shift+minus sign`
